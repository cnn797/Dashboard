# Dashboard

A personal dashboard hosted on GitHub Pages.

## Live Site

Once deployed, your site will be live at:
`https://cnn797.github.io/dashboard/`

---

## How to Edit

1. Open `index.html` in any text editor (VS Code recommended).
2. Make your changes and save the file.

---

## How to Commit and Push Changes

After editing `index.html`, run these commands in Terminal from the `Dashboard` folder:

```bash
git add index.html
git commit -m "Update dashboard"
git push
```

GitHub Pages will automatically redeploy within ~30 seconds to 2 minutes.

---

## How GitHub Pages Auto-Deploy Works

- GitHub Pages watches the `main` branch of your repo.
- Every time you `git push`, GitHub detects the change and rebuilds the site.
- No manual upload or build step is needed — it serves `index.html` directly from the root.
